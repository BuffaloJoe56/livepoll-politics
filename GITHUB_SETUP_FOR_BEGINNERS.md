# GitHub Setup Guide for Complete Beginners

This file is written for someone who has never used GitHub before.

## What is GitHub? (Simple Explanation)

GitHub is like a **cloud storage + version history** for your code.

- It lets you save your project online.
- It keeps a history of every change you make.
- It allows tools like Netlify (and later Firebase) to automatically update your website when you make changes.

Think of it like Google Drive, but made specifically for code and websites.

## Why Do We Need It?

Right now you are using "Netlify Drop" (dragging the folder).  
This works, but every time you want to update your site you have to rebuild and drag again.

With GitHub + proper Netlify:
- You make changes on your computer.
- You "push" the changes once.
- Your website updates automatically.

This will also make moving to Firebase later much easier.

## Step-by-Step: Create GitHub Account (First Time)

1. Open your browser and go to: https://github.com

2. Click the green **"Sign up"** button (top right).

3. Fill in:
   - Email address
   - Create a password
   - Choose a username (this will be part of your future website links sometimes)

4. Verify your email address when GitHub sends you the code.

5. You now have a GitHub account.
